# Volcano Diagrams

This folder contains scientific diagrams and educational illustrations for the Ring of Fire book.

## Diagram Types Needed:

### Tectonic Settings
- Subduction zone cross-sections
- Divergent boundary diagrams
- Hotspot/mantle plume diagrams
- Earth's internal layers

### Volcano Types
- Volcano anatomy (labeled cross-section)
- Shield vs stratovolcano comparison
- Caldera formation sequence

### Eruption & Magma
- Eruption types comparison (Hawaiian to Plinian)
- VEI scale visual
- Magma types (basalt, andesite, rhyolite)
- Volcanic cycle diagram

## Creating Diagrams

### Free Tools:
1. **Excalidraw** (https://excalidraw.com/) - Hand-drawn style, great for cross-sections
2. **Draw.io** (https://app.diagrams.net/) - Professional diagrams
3. **Canva** (free tier) - Charts and infographics

### Public Domain Sources:
- USGS educational materials
- NASA visualizations
- Wikimedia Commons
