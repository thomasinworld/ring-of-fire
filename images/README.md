## Image Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `mt-st-helens-eruption-1980.jpg`
- ✅ `shield-volcano-cross-section.png`
- ❌ `IMG_1234.jpg`
- ❌ `Volcano Photo.JPG`

## File Formats

- **Photos**: JPG (smaller file size)
- **Diagrams/illustrations**: PNG (better for graphics with text)
- **Keep files under 2 MB each** for faster loading
